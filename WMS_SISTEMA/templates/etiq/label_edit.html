Editor guiado

Este modo foi simplificado para quem precisa alinhar a etiqueta rapidamente sem entender medidas técnicas.

1
Escolha a parte da etiqueta Selecione cabeçalho, textos, cor ou código.
2
Ajuste com setas Use movimentos curtos até o encaixe ficar bom na prévia.
3
Salve como modelo Guarde o layout no navegador ou no servidor para reutilizar.
