🚀 INSTALAÇÃO E USO

1️⃣ INICIAR O SERVIDOR
   Execute o arquivo: INICIAR_WMS_SERVIDOR.bat
   
   O servidor iniciará automaticamente na porta 5000

2️⃣ ACESSAR O SISTEMA
   
   ➜ No próprio servidor:
      http://localhost:5000
      http://192.168.1.210:5000
   
   ➜ De outros computadores na rede:
      http://192.168.1.210:5000

3️⃣ CREDENCIAIS PADRÃO
   Usuário: admin
   Senha: admin

   ⚠️ IMPORTANTE: Altere a senha após o primeiro acesso!

=================================================================
🌐 CONFIGURAÇÃO DE REDE
=================================================================

Para que outros computadores acessem o sistema:

1. Certifique-se de que o servidor está rodando
2. Libere a porta 5000 no Firewall do Windows:
   - Painel de Controle
   - Firewall do Windows
   - Configurações Avançadas
   - Regras de Entrada
   - Nova Regra → Porta → TCP 5000

3. Compartilhe o endereço com os usuários:
   http://192.168.1.210:5000

=================================================================
❓ SOLUÇÃO DE PROBLEMAS
=================================================================

⚠️ Erro "Porta já em uso"
   → Outro programa está usando a porta 5000
   → Feche outros servidores ou reinicie o computador

⚠️ Não consigo acessar pela rede
   → Verifique o firewall
   → Confirme que o IP 192.168.1.210 está correto
   → Teste ping 192.168.1.210 no cmd

⚠️ Erro no banco de dados
   → Verifique se wms_database.mdb existe
   → Confirme permissões de leitura/escrita
