================================================================================
DISTRIBUIÇÃO DO WMS PARA PCs SEM PYTHON
================================================================================

✓ EXECUTÁVEL CRIADO COM SUCESSO!
✓ DETECÇÃO AUTOMÁTICA DE IP - Funciona em QUALQUER rede

Localização: dist\WMS_Server\

================================================================================
COMO DISTRIBUIR:
================================================================================

1. Copie TODA a pasta "WMS_Server" para o PC destino
   (pode ser pen drive, rede compartilhada, etc.)

2. No PC destino (servidor), execute um destes:
   - INICIAR_WMS.bat  (mais fácil)
   - WMS_Server.exe   (direto)

3. O servidor DETECTA AUTOMATICAMENTE o IP e mostra:
   "Ou via rede: http://[IP_AUTOMATICO]:5000"
   
   Exemplos:
   - Rede 192.168.1.x → http://192.168.1.50:5000
   - Rede 192.168.2.x → http://192.168.2.28:5000
   - Rede 10.0.0.x → http://10.0.0.100:5000

4. Nos outros PCs (clientes), abra o navegador e use o IP mostrado:
   http://[IP_DO_SERVIDOR]:5000

================================================================================
ESTRUTURA DA PASTA (NÃO SEPARAR):
================================================================================

WMS_Server/
├── INICIAR_WMS.bat      ← Duplo clique aqui para iniciar
├── WMS_Server.exe       ← Executável principal
├── wms_database.mdb     ← Banco de dados
├── templates/           ← Interface web (pasta completa)
├── _internal/           ← Bibliotecas (NÃO MEXER)
└── COMO_USAR.txt        ← Instruções detalhadas

================================================================================
REQUISITOS DOS PCs:
================================================================================

PC SERVIDOR (onde roda o WMS_Server.exe):
  ✓ Windows 7 ou superior
  ✓ NÃO precisa de Python
  ✓ Precisa ter a pasta WMS_Server completa

PCs CLIENTES (só acessam):
  ✓ Qualquer sistema (Windows, Mac, Linux, Android, iOS)
  ✓ Navegador web (Chrome, Firefox, Edge, Safari)
  ✓ Mesma rede do servidor (192.168.2.x)

================================================================================
TESTADO E FUNCIONANDO:
================================================================================

✓ Conecta ao banco MDB
✓ Inicia servidor na porta 5000
✓ Aceita conexões de outros PCs
✓ Funciona sem Python instalado
