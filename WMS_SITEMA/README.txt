## Como Usar

1. Execute **WMS_Server.exe**
2. O servidor iniciará na porta 5002
3. Acesse pelo navegador: http://localhost:5002
4. Faça login com suas credenciais

## Credenciais Padrão
- **Usuário:** admin
- **Senha:** admin
